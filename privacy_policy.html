PassaFacile 隐私政策

最后更新日期：2025年10月1日
本隐私政策说明了PassaFacile应用如何收集、使用和保护您的个人信息。

1. 信息收集

我们收集以下类型的信息：

  • 账户信息：邮箱地址、用户名
  • 学习数据：练习记录、错题记录、学习进度
  • 设备信息：设备类型、操作系统版本
  • 使用数据：应用使用时长、功能使用频率
  • IP 地址：用于地理位置判断，自动选择最优 API 服务器。IP 地址不会被存储或上传到服务器。

2. 信息使用

我们使用收集的信息用于：

  • 提供个性化学习体验
  • 跟踪学习进度和成绩
  • 改进应用功能和用户体验
  • 处理内购交易
  • 提供客户支持

2.5 IP 地址检测（不使用GPS定位）

为了提供最佳的应用性能，我们会在应用启动时检测您的 IP 地址以判断您的国家/地区。我们不会获取您的GPS位置信息或精确地理坐标。

  • 检测方式：通过第三方 IP 地理位置服务 (ipapi.co, ip-api.com, Cloudflare) 仅检测国家代码（中国/非中国）
  • 检测内容：IP 地址对应的国家代码（如 CN、IT 等）
  • 使用目的：自动选择最优 API 服务器地址，为中国用户提供反向代理服务以提升连接速度
  • 数据处理：IP 地址不会被存储或上传到我们的服务器，仅在本地用于判断国家/地区
  • 缓存机制：检测结果会被缓存，避免重复查询
  • 不使用GPS：本应用不会请求或使用任何GPS定位权限，不会获取您的精确位置坐标

3. 信息共享

我们不会向第三方出售、交易或转让您的个人信息，除非：

  • 获得您的明确同意
  • 法律要求或政府机关要求
  • 保护我们的权利、财产或安全

4. 数据存储

您的数据存储在Supabase云数据库中，采用行业标准的安全措施保护。数据传输使用SSL加密。

5. 用户权利

您有权：

  • 访问您的个人数据
  • 更正不准确的数据
  • 删除您的账户和数据
  • 导出您的学习数据

6. 内购信息

应用内购买通过Apple App Store处理。我们不存储您的支付信息。订阅管理请通过Apple ID设置进行。

7. 儿童隐私

我们的服务面向13岁以上用户。我们不会故意收集13岁以下儿童的个人信息。

8. 政策更新

我们可能会更新此隐私政策。重大变更将通过应用内通知告知用户。
